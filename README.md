How to launch a game ?

Open console in folder and type .\Chip8Emulator.exe "VideoScale" "TimeScale" .\games\<Rom>.ch8

Example : .\Chip8Emulator.exe 10 1 .\games\Tank.ch8

---------------
Keys : Escape, X, 1, 2, 3, Q, W, E, A, S, D, Z, C, 4, R, F, V.
